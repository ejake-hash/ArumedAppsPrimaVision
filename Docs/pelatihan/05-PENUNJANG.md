# 05 — Panduan Penunjang

← Kembali ke [Modul Pelatihan](00-MODUL-PELATIHAN-PRIMA-VISION.md)

---

## Untuk siapa & kapan dipakai

Untuk **petugas pemeriksaan penunjang** (mis. Biometri, OCT, USG mata). Di sini Anda menerima order pemeriksaan dari dokter, mengerjakannya, dan mengirim hasilnya kembali ke dokter. Buka lewat menu **Penunjang** di sidebar.

> **Penunjang** = pemeriksaan tambahan untuk membantu dokter menegakkan diagnosa. **Biometri** = pengukuran mata untuk menentukan kekuatan lensa tanam (IOL) pada operasi katarak.

---

## Tata letak layar

Di bagian atas ada **dua tab modul**:
- **Antrean** — pekerjaan harian: daftar order dari dokter.
- **Jenis Penunjang** — master daftar jenis pemeriksaan (biasanya dikelola admin).

Pada tab **Antrean**:
- **Panel kiri:** antrean pasien yang punya order penunjang (saringan Semua / BPJS / Umum-Asuransi).
- **Panel kanan:** daftar order pasien terpilih + panel input hasil.

---

## Mengerjakan order

1. Pilih pasien dari antrean. Akan tampil daftar order yang diminta dokter (satu pasien bisa punya beberapa order).
2. Pada order yang dikerjakan, klik **Mulai** untuk membuka panel input hasil order tersebut.
3. Isi hasil:
   - **Pemeriksaan Biometri:** form khusus per mata — **OD (Mata Kanan)** dan **OS (Mata Kiri)**: nilai K, panjang sumbu (AXL), dan ukuran terkait. Dari sini sistem dapat membantu **rekomendasi IOL**.
   - **Pemeriksaan lain:** isi **Kesimpulan** dan **Ringkasan/Catatan** (mis. "dalam batas normal").
4. **Lampiran (opsional):** unggah file hasil alat (mis. PDF/gambar) bila ada.
5. Simpan. Hasil bisa disimpan sebagai draf lebih dulu, lalu diselesaikan.

### Menyelesaikan
Setelah semua order selesai diisi, selesaikan pemeriksaan. Pasien **otomatis kembali** ke antrean dokter (ditandai "Selesai Penunjang") untuk dilanjutkan.

---

## Kesalahan umum & tips

- **Klik "Mulai"** pada order yang dikerjakan untuk membuka panel input — panel hanya terbuka satu order pada satu waktu.
- **OD = kanan, OS = kiri** pada form Biometri — jangan tertukar.
- **Selesaikan semua order** sebelum mengakhiri, agar pasien lengkap saat kembali ke dokter.
- Tab **Jenis Penunjang** untuk menambah/mengubah daftar jenis pemeriksaan — biasanya hanya dipakai admin, bukan pekerjaan harian.
